<h3>Dotenv</h3>
<b>Create an .env file in root directory and you need to complete the following information related to the database:</b> 

```bash
- DB_USERNAME=the username 
- DB_PASSWORD=the password 
- DB_HOST= the hostname 
- DB_PORT=the port 
- DB_NAME= the name 
```
